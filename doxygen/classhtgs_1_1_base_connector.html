Detailed Description

Parent class for Connector, which removes the template type of the Connector.

Used within data structures to hold various types of Connectors. Each Connector is built using TaskGraph routines to add ITasks to a TaskGraph.

Each Connector holds onto a priority queue that acquires/distributes IData from a producer/consumer ITask.

Most common use for this class is to indicate when the producer for this Connector has finished pushing data onto its priority queue.

inputConnector->isInputTerminated();
